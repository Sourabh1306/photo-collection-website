# 📸 Photo Collection Website

This is a simple web project where users can upload and organize images into albums. It is built using HTML, CSS, and JavaScript, and stores data in the browser's local storage.

## Features
- Create albums
- Upload multiple images
- View gallery
- Delete images
- Responsive design

## Tech Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created as a college assignment to practice web development and create a real-world website.
